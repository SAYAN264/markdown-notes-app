# Backend

This directory contains the backend server for the Notes Vite project.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the backend server:
   ```sh
   node index.js
   ```

## Files
- `index.js`: Main backend server file.
- `package.json`: Backend dependencies and scripts.
